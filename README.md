# HTML5 Canvas 2d RPG Game Engine

A game engine for 2d RPG game working through HTML5 Canvas and Javascript 

## Getting Started

Copy code to website and open index.html

### Prerequisites

None

### Installing

Copy contents of game folder to webserver.
